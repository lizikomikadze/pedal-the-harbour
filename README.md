# Pedal the Harbour — Digital Guidebook

Modern, desktop-first, minimal travel-app style website for:
**“Pedal the Harbour: A One-Day Sustainable Cycling Guide from Tai Wai to Tai Po Waterfront in Hong Kong.”**

## Run locally

From this folder:

```bash
cd website
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Customize content

- **Stops + route**: edit the `STOPS` array in `main.js` (names, coordinates, images, text, distances).
- **Look & feel**: edit `styles.css`.
